# What this example shows

This example shows how to deploy a VPC with a private and public subnet spanned over two availability zones.

## Basic usage

See the code in [main.tf] to see the implementation details.

## Running the example

### Cloning the repository

```bash
git clone https://github.com/mineiros-io/terraform-aws-vpc.git
cd terraform-aws-vpc/examples/complete-example
```

### Initializing Terraform

Run `terraform init` to initialize the example and download providers and the module.

### Planning the example

Run `terraform plan` to see a plan of the changes.

### Applying the example

Run `terraform apply` to create the resources.
You will see a plan of the changes and Terraform will prompt you for approval to actually apply the changes.

### Destroying the example

Run `terraform destroy` to destroy all resources again.
